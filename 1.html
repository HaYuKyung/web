HTML이란 무엇인가?

Hypertext Markup Language(HTML) is the standard markup language for creating web pages and web applications. Web browers receive HTML documents from a web werver or from local strorage and render them into multimedia web pages. HTML describes the structure of a web page semantically and originally included cues for the appearance of the document.
